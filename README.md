# SmartAnaly-Cloud-Intelligent-Student-Grade-Management-System-with-AI-Natural-Language-Query
传统学生成绩管理系统只有固定菜单查询（按学号、按姓名），操作死板。 本项目结合 AI 大模型开放 API 接口 + 本地 SQLite 数据库，实现： 常规成绩增删改查管理 + AI 自然语言智能查询数据 用户不用写 SQL，不用记菜单，直接用口语提问，AI 自动生成 SQL 语句，程序执行并返回统计分析结果。 属于传统管理系统升级智能化，难度中等偏上，技术全面，复杂度适中，不会太难做不完，又远高于普通纯控制台管理系统。
