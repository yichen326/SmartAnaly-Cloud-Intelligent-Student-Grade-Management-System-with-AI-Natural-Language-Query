# 智析云途 - AI学生成绩管理系统

基于火山方舟豆包大语言模型的学生成绩智能管理系统，支持文本指令交互、数据可视化编辑、AI智能分析。

## 学号标准

系统采用 **8位数字学号格式：GGCCNNNN**

| 部分 | 位数 | 说明 | 范围 |
|------|------|------|------|
| GG | 2位 | 年级代码 | 01=高一, 02=高二, 03=高三 |
| CC | 2位 | 班级编号 | 01~30 |
| NNNN | 4位 | 顺序号 | 0001~9999 |

**示例：**
- `01010001` → 高一1班第1号学生
- `02020015` → 高二2班第15号学生
- `03150088` → 高三15班第88号学生

**验证规则：**
- 必须为8位纯数字
- 年级代码必须为01/02/03
- 班级编号必须在01~30之间
- 顺序号必须在0001~9999之间
- 添加重复学号会报错

## 功能特性

### 📊 核心管理功能
- **学生增删改查** — 支持通过文本指令或表格界面操作
- **成绩编辑** — 表格内联双击编辑，自动计算总分
- **学号验证** — 严格的8位学号格式校验，错误添加直接报错
- **数据导入导出** — 支持Excel导出，TXT自动同步供AI读取
- **筛选排序** — 表格支持按年级、班级筛选，按学号/总分/班级排序

### 🏆 排名系统
- **班级排名** — 支持按总分或单科在班级内排名
- **年级排名** — 支持按总分或单科在年级内排名
- **全校排名** — 全校范围内的成绩排名

### 🤖 AI智能分析
- **班级成绩分析** — 分析各科平均分、最高/最低分、及格率、优秀率
- **年级成绩分析** — 分析年级整体成绩，含各班级对比
- **单科详细分析** — 分数段分布、优秀/良好/及格/不及格名单
- **AI自由问答** — 非固定指令由AI基于数据库内容智能回答
- **人性化点评** — 每次操作后AI生成温暖、有温度的小评

### 💬 自然语言指令
支持模糊匹配和自然语言理解，例如：
- "显示所有学生"
- "分析高一1班成绩"
- "高一1班怎么样"
- "1班数学排名"
- "修改张三数学成绩95"
- "给点学习建议"
- "今天心情不好"
- 以及更多自由对话...

## 快速开始

### 1. 安装依赖

```bash
pip install PyQt5 openpyxl requests
```

### 2. 配置API

在 `main.py` 顶部修改以下配置：

```python
ARK_API_KEY = "your-api-key-here"         # 火山方舟API密钥
EP_ID = "your-endpoint-id-here"            # 大模型端点ID
```

### 3. 运行系统

```bash
python main.py
```

默认账号：`admin` 密码：`123456`

### 核心类说明

| 类名 | 职责 |
|------|------|
| `DoubaoAI` | 火山方舟API调用、指令意图解析、自然语言理解 |

| `DataTableDialog` | 数据表格GUI、内联编辑、筛选排序、统计显示 |
| `AIWorker` | 后台线程处理AI请求、二次确认 |

| `LoginDialog` | 登录认证 |

### 扩展建议

1. **添加更多科目**：在 `subjects` 列表中添加科目名，扩展 `score` 表结构
2. **多用户支持**：扩展 `user` 表，增加权限管理
3. **图表可视化**：集成 `matplotlib` 或 `pyecharts` 展示成绩分布图
4. **批量导入**：支持从Excel/CSV批量导入学生数据
5. **成绩跟踪**：记录历次考试成绩变化，生成进步/退步趋势
6. **家长通知**：集成短信或邮件通知，发送成绩报告
7. **教师评价**：增加评语字段，支持教师批量点评
8. **考试管理**：支持多场考试数据管理，对比分析
9. **导出PDF报告**：生成带图表的学生成绩报告单
10. **自动备份**：定时备份数据库到云存储或本地目录
11. **语音交互**：集成语音识别，支持语音指令
12. **Web界面**：使用Flask/FastAPI搭建Web管理后台
13. **API接口**：提供RESTful API供第三方系统调用
14. **离线模型**：部署本地小模型处理简单指令，降低API成本
