## 🔢 学号标准

系统采用 **8位数字学号格式：GGCCNNNN**，严格校验，错误添加直接报错。

| 部分 | 位数 | 说明 | 范围 |
|:---:|:---:|------|:----:|
| **GG** | 2位 | 年级代码 | `01`=高一, `02`=高二, `03`=高三 |
| **CC** | 2位 | 班级编号 | `01` ~ `30` |
| **NNNN** | 4位 | 顺序号 | `0001` ~ `9999` |

**示例：**
| 学号 | 解析 |
|------|------|
| `01010001` | 高一1班第1号学生 |
| `02020015` | 高二2班第15号学生 |
| `03150088` | 高三15班第88号学生 |

**验证规则：**
- ✅ 必须为 **8位纯数字**
- ✅ 年级代码必须为 `01` / `02` / `03`
- ✅ 班级编号必须在 `01` ~ `30` 之间
- ✅ 顺序号必须在 `0001` ~ `9999` 之间
- ✅ 重复学号自动拒绝并报错

## 🚀 核心功能

### 📊 核心管理功能

| 功能 | 描述 |
|------|------|
| **学生增删改查** | 支持通过聊天指令或表格界面操作 |
| **成绩编辑** | 表格内双击编辑，自动计算总分并保存排名 |
| **学号验证** | 严格的8位学号格式校验，格式错误/重复添加直接报错 |
| **筛选排序** | 按年级、班级筛选，按学号/总分/班级排序 |
| **数据监控** | 状态栏实时显示学生总数、平均分、排名状态 |

### 🏆 排名系统

| 排名类型 | 描述 |
|----------|------|
| **班级排名** | 班级内总分/单科排名，显示等级与平均分 |
| **年级排名** | 全年级总分排名，跨班级对比 |
| **全校排名** | 全校范围内的总分排名 |

> 每次数据变更后自动更新所有排名。

### 🤖 AI智能分析

| 分析维度 | 内容 |
|----------|------|
| **班级分析** | 各科平均分、最高/最低分、及格率、优秀率、等级分布图 |
| **年级分析** | 年级整体概览、各班级对比、年级均分 |
| **学生评估** | 个人总分、均分、等级、优劣势科目分析、个性化评语 |
| **自由问答** | 非固定指令由DeepSeek AI基于数据库上下文智能回复 |
| **人性化点评** | 每次分析后AI生成温暖有温度的小评语 |

### 💬 AI自然语言交互（核心亮点）

采用 **DeepSeek V4 Flash 大模型** 驱动的智能对话引擎，支持五种复杂交互场景：

| 场景类型 | 示例 | AI处理方式 |
|----------|------|------------|
| **日常对话** | "今天天气真好"、"心情不好"、"你好"、"辛苦了" | AI像朋友一样温暖回应，不触发系统功能 |
| **复合问题** | "高一1班怎么样？高三2班呢？"、"分析张三和李四的成绩" | AI一次性回答所有问题，不遗漏任何一个 |
| **比较分析** | "张三和李四谁成绩好"、"对比高一1班和高一2班" | AI直接基于系统数据分析比较 |
| **模糊表达** | "帮我分析分析"、"最近怎么样"、"我们班" | AI主动引导，推算出最可能的意图 |
| **精确指令** | "显示所有学生"、"总分排名"、"分析高一1班" | AI识别并触发对应系统功能 |

支持模糊匹配、拼音识别和自然语言理解，例如：

```
"显示所有学生"
"分析高一1班成绩"
"高一1班怎么样"
"gao yi 1 ban pai ming"
"张三的成绩"
"修改张三数学成绩95"
"给点学习建议"
"今天心情不好"
"系统统计"
```

## ⚡ 快速开始

### 1️⃣ 环境要求

- Python 3.8+
- 操作系统：Windows / Linux / macOS

### 2️⃣ 安装依赖

```bash
pip install PyQt5 openpyxl requests
```

### 3️⃣ 配置DeepSeek API

如需使用AI智能分析功能，`config.json` 中已预置API密钥（或替换为你自己的密钥）：

```json
{
    "deepseek": {
        "api_key": "sk-032f75440f6542d690f9ccc523b01cd3",
        "api_url": "https://api.deepseek.com/v1/chat/completions",
        "model": "deepseek-v4-flash"
    }
}
```

> 💡 当前使用 **DeepSeek V4 Flash** 模型，快速且智能。
> 💡 不配置API密钥时，系统仍可使用所有基础功能，AI回复将使用本地回退逻辑。

### 4️⃣ 准备学生数据（可选）

创建 `students_data.txt` 文件，格式如下（一行一名学生，`|` 分隔）：

```
学号|姓名|性别|语文|数学|英语|物理|化学|生物
01010001|张三|男|78|85|92|67|88|73
01010002|李四|女|92|88|96|85|90|87
...
```

> 系统启动时自动读取该文件并导入数据库，格式错误/学号违规的行会自动跳过并提示。

### 5️⃣ 启动系统

```bash
python main.py
```

**默认登录账号：**
| 字段 | 默认值 |
|------|--------|
| 用户名 | `admin` |
| 密码 | `123456` |

## 📦 打包发布

### 一键打包（推荐 Windows）

```bash
build.bat
```

自动完成环境检测、依赖安装、PyInstaller打包，输出到 `dist\智析云途.exe`。

### 手动打包

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "智析云途" ^
    --add-data "config.json;." ^
    --add-data "students_data.txt;." ^
    --hidden-import "requests" ^
    --hidden-import "openpyxl" ^
    --hidden-import "PyQt5" ^
    --hidden-import "PyQt5.QtWidgets" ^
    --hidden-import "PyQt5.QtCore" ^
    --hidden-import "PyQt5.QtGui" ^
    --clean --noconfirm main.py
```

### 打包参数说明

| 参数 | 说明 |
|------|------|
| `--onefile` | 单文件模式，所有依赖打包到一个exe |
| `--windowed` | 无控制台窗口，适配GUI应用 |
| `--add-data` | 附带数据文件（config.json, students_data.txt等） |

### 打包后部署

将 `dist\` 目录下的以下文件一同分发：

```
dist/
├── 智析云途.exe          # 主程序
├── config.json           # 配置文件（需填写API Key）
└── students_data.txt     # 初始数据（可选）
```

> 💡 首次运行时，系统会自动创建 `school_final.db` 数据库文件。

### 数据表结构

```sql
CREATE TABLE students (
    id           INTEGER PRIMARY KEY AUTOINCREMENT,
    student_id   TEXT UNIQUE NOT NULL,        -- 8位学号 GGCCNNNN
    name         TEXT NOT NULL,               -- 姓名
    gender       TEXT CHECK(gender IN ('男','女')),  -- 性别
    grade_code   TEXT NOT NULL DEFAULT '01',  -- 年级代码 (01/02/03)
    class_code   TEXT NOT NULL DEFAULT '01',  -- 班级代码 (01~30)
    chinese      REAL DEFAULT 0,              -- 语文
    math         REAL DEFAULT 0,              -- 数学
    english      REAL DEFAULT 0,              -- 英语
    physics      REAL DEFAULT 0,              -- 物理
    chemistry    REAL DEFAULT 0,              -- 化学
    biology      REAL DEFAULT 0,              -- 生物
    total_score  REAL DEFAULT 0,              -- 总分
    class_rank   INTEGER DEFAULT -1,          -- 班级排名
    grade_rank   INTEGER DEFAULT -1,          -- 年级排名
    created_at   TIMESTAMP DEFAULT CURRENT_TIMESTAMP  -- 创建时间
);
```
